# IELTS Pro Website

Website luyện thi IELTS với 8 volumes Cambridge IELTS.

## Tính năng
- 8 Volumes với 10 tests mỗi volume
- Listening với audio player
- Reading với highlight text
- Writing với word counter
- Speaking practice
- Theo dõi tiến độ học tập

## Cách sử dụng
1. Mở `index.html` trong trình duyệt
2. Chọn Volume muốn luyện tập
3. Làm bài và xem kết quả
